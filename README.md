# Workout App " Power Hour "


## OVERVIEW

A workout tracking app that allows the user to enter workouts they have completed.

### Technologies Used

- JWT
- MongoDB
- Mongoose
- Express JS
- Node JS
- HTML5

- React

- Navigate the app with a navbar
- Add workouts
- View all of my saved workouts
